# Components

We provide a set of pre-built components to help you build your UI faster.

## Button

The Button component is used to trigger actions.

```tsx
import { Button } from 'my-library';

<Button>Click me</Button>
```

## Card

Cards are used to group related content.

```tsx
import { Card } from 'my-library';

<Card>
  <h2>Title</h2>
  <p>Content</p>
</Card>
```
