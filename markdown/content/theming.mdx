# Theming

Our library supports full theming capabilities.

## Customizing Colors

You can override the default colors in your CSS or configuration.

```css
:root {
  --primary-color: #ff0000;
}
```

## Dark Mode

Dark mode is supported out of the box. Just toggle the class on the body element.
