# Configuration

Create a `config.ts` file in the root of your project.

```ts
export default {
  theme: 'dark',
  plugins: [],
}
```

## Options

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| theme | string | 'light' | The color theme |
| plugins | array | [] | List of plugins |
