# How It Works

The skill installs a set of markdown files into your project that AI agents can read. Each rule has a brief explanation, an incorrect code example, and a correct one. Agents use these as guardrails when generating or reviewing code.

The structure follows the [Vercel agent-skills](https://github.com/vercel-labs/agent-skills) pattern:

- `SKILL.md` — compact quick-reference with one-liner per rule
- `AGENTS.md` — full compiled document with all rules expanded
- `rules/` — individual rule files for granular access

# Usage

I recommend using it on a case-by-case basis rather than having it always on. One example would be reviewing your animations, another would be implementing audio feedback on your site.

The skill file is available on [GitHub](https://github.com/raphaelsalaja/userinterface-wiki/tree/main/skills).
