Inspired by [Benji's experiments with Claude](https://benji.org/morphing-icons-with-claude). Every icon uses exactly three SVG lines. Icons that need fewer collapse the extras to invisible points. This means any icon can morph into any other since they share the same underlying structure.[^1]

[^1]: [Benji's experiments with Claude](https://benji.org/morphing-icons-with-claude)
